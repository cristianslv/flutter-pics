# Flutter Pics

#### This project is small picture gallery built with Flutter.

```bash
$ flutter --version

Flutter 2.2.1 
Tools • Dart 2.13.1
```


Make sure to install all Project/Dev dependencies.

```bash
$ flutter pub get
```


- First of all, you need to run the app either on your smartphone or on your simulator →

```bash
$ flutter run
```
